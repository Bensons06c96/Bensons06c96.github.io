步驟五：將公鑰添加到 GitHub

  1. 複製公鑰內容：

  2. 登入 GitHub 網站

  3. 點擊右上角你的頭像 → 「Settings」

  4. 在左側選單中點擊「SSH and GPG keys」

  5. 點擊「New SSH key」按鈕

  6. 在「Title」欄位中輸入一個有意義的名稱，例如「我的工作電腦」

  7. 在「Key」欄位中粘貼你剛才複製的公鑰內容

  8. 點擊「Add SSH key」按鈕儲存
